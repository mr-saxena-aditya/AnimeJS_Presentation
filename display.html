Display: block

Block elements take up the full width available and start on a new line.

Display: inline

Inline elements only take up as much width as necessary and do not start on a new line.

Display: inline-block

Inline-block elements are similar to inline elements but can have margins and paddings applied.

Display: flex

Flex elements are flexible containers that can be easily aligned and arranged.

Try resizing the window to see the flex items wrap when the container width is limited.

Display: inline-flex

Inline-flex elements are similar to flex elements but do not start on a new line.

This inline-flex container is centered both horizontally and vertically.

Display: grid

Grid elements are powerful containers for creating layout structures using rows and columns.

This is a grid with three columns.

Display: inline-grid

Inline-grid elements are similar to grid elements but do not start on a new line.

This is an inline-grid with three columns.

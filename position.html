Position: static

This is the default positioning value for elements. It follows the normal document flow and is not affected by the top, bottom, left, or right properties.

Position: relative

The element is positioned relative to its normal position in the document flow. It can be moved using the top, bottom, left, or right properties without affecting other elements' positions.

Child elements with position: absolute will be positioned relative to this element.

Position: absolute

The element is taken out of the normal document flow and positioned relative to its closest positioned ancestor (or the viewport if no positioned ancestor is found).

It doesn't leave a space in the document flow, and other elements ignore it.

Position: fixed

The element is positioned relative to the viewport and stays fixed at its specified position even when scrolling the page.

It doesn't leave a space in the document flow and is always visible regardless of scrolling.
